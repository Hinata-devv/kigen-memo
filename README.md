# 🍓 きげんmemo

食品の期限をシンプルに管理できるWebアプリです。  
冷蔵庫の中身や開封後の期限を忘れずに管理できます。

---

## 🔗 デモURL
https://hinata-dev.github.io/kigen-memo/

※スマホからの利用を想定しています

---

## 🧩 主な機能

- 📅 食品の期限登録
- 🗂 カテゴリ分け（調味料 / ドレッシング / 冷蔵系 など）
- ⏰ 期限の自動カウント（◯日以内）
- 🔍 フィルター機能（カテゴリ別表示）
- 🔔 通知機能（※一部環境で制限あり）
- 📸 カメラでの入力補助（OCR）

---

## 💡 こだわりポイント

- シンプルで直感的に使えるUI設計
- スマホでの操作性を重視
- 視覚的に分かりやすい期限表示
- 柔らかい配色で日常的に使いやすいデザイン

---

## ⚠️ 注意点

- Googleログインは一部環境（特にアプリ内ブラウザ）で利用できない場合があります  
  → その場合はSafariやChromeなどのブラウザでご利用ください

---

## 🛠 使用技術

- HTML / CSS / JavaScript
- Firebase（Authentication / Firestore）
- Tesseract.js（OCR）
