# 🍓 きげんmemo

## 📱 アプリ概要
調味料や粉物などの賞味期限を管理するスマホ向けWebアプリです。

## 🧠 作った背景
調味料や小麦粉が気づかないうちに期限切れになることが多く、
簡単に管理できるアプリを作成しました。

## ✨ 主な機能（v1.0）
- 調味料の登録
- 開封日・期限の入力
- データ保存（localStorage）
- 期限順表示
- 残り日数表示
- 期限切れ表示
- 色分け表示
- 件数サマリー
- 削除 / 全削除

## 🛠 使用技術
- HTML
- CSS
- JavaScript
