# 🍓 きげんmemo

食品の賞味期限をシンプルに管理できるWebアプリです。  
開封日・期限・カテゴリごとに整理し、期限切れや残り日数を分かりやすく確認できます。

---

## ✨ 主な機能

- 📅 期限日と残り日数の自動計算
- ⚠️ 期限切れ・3日以内・7日以内の色分け表示
- 🗂 カテゴリごとの管理（調味料 / ドレッシング / 粉もの / 冷蔵系）
- 🔍 商品名での検索
- 🧹 ワンクリックでリストをリセット
- 📷 写真から期限候補を抽出（OCR機能）

## 💡 工夫したポイント

- 直感的に使えるUI設計（入力 → 一覧 → 削除の流れ）
- 期限が近いものが一目で分かる色分け
- カテゴリと検索で整理しやすい構成
- OCRは「候補提示」に留め、誤認識のストレスを軽減
