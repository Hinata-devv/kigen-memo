# 🍓 きげんmemo

食品の賞味期限を直感的に管理できるWebアプリです。  
撮影・入力・確認までをシンプルな操作で完結できます。 
---

### ✨ 主な機能

- 📷 写真から期限候補を抽出（OCR / 自動入力対応）
- 📅 期限日と残り日数の自動計算
- ⚠️ 期限切れ・3日以内・7日以内の色分け表示
- 🗂 カテゴリごとの管理（調味料 / ドレッシング / 粉もの / 冷蔵系）
- 🔍 商品名で検索
- ♻️ ワンクリックでリストをリセット
